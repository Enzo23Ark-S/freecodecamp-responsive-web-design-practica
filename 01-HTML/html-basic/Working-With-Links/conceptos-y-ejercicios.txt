What Is the Difference Between Absolute and Relative Paths?

Resumen: Una ruta es una cadena que especifica la ubicación de un archivo o directorio en un sistema de archivos. Una ruta absoluta también incluye el protocolo, que podría ser http, https, y file y el nombre de dominio si el recurso está en la web.

What Is the Difference Between Slashes, a Single Dot, and Double Dot in Path Syntax?

Resumen: Las rutas de archivos hay tres, Primero está la barra, que puede ser una barra invertida (\) o barra diagonal (/) dependiendo de su sistema operativo. El segundo es el punto único (.). Y por último tenemos el doble punto (..).

What Are the Different Link States, and Why Are They Important?

Resumen: Nuevos valores para saber si un enlace ya haya sito visitado
El hover es un estado que se aplica cuando un usuario pasa el cursor sobre un enlace. Este estado es útil para brindar atención adicional a un enlace, para garantizar que un usuario realmente tenga la intención de hacer clic en él.
focus. Este estado se aplica cuando nos centramos en un enlace.
active. Este estado se aplica a los enlaces que están siendo activados por el usuario.

                    Preguntas

¿Qué estado se aplica cuando un usuario hace clic en el enlace?

    :active

    link, visited, hover, active.
